# 🎓 AI College Finder

An AI-powered web application that helps students and professionals discover suitable universities based on their academic profile. The application provides personalized recommendations using Google's Gemini AI, allows university search, compares universities, saves favorites, and generates a professional PDF report.

---

## 🚀 Features

* 👤 Student Profile Management
* 🔎 Search Universities by Country or Name
* 🤖 AI-Powered University Recommendations
* ⚖️ Compare Two Universities with AI
* ⭐ Save Favorite Universities
* 📄 Export Professional PDF Report
* 🌐 Clean and Responsive Streamlit Interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* Requests
* ReportLab

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/AI-College-Finder.git
cd AI-College-Finder
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Get a Gemini API Key

Create a free API key from:

https://aistudio.google.com/apikey

---

## ▶️ Run the Application

```bash
streamlit run streamlit_app.py
```

The application will automatically open in your browser.

## 📖 How to Use

1. Fill in your student profile.
2. Search universities by country or university name.
3. Generate AI recommendations.
4. Compare two universities.
5. Save your favorite universities.
6. Export your complete report as a PDF.
